# ✈3D SkyFlight

[비행 체험하러 가기](https://sky-flight.vercel.app/)
✈🛬🛫Blender를 활용해 지면 모델을 생성하는 과정을 이해하고 비행기 모델을 통한 비행 시뮬레이션 구현 연습

---

### 🎮조작법

- 상승(`W`)
- 하강(`S`)
- 좌향(`A`)
- 우향(`D`)
- 다시하기(`R`)
- 터보(`Shift`)

---

### 🏁목표

모든 링을 통과해 비행 능력을 입증하자!
